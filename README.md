# StockVison

![Logo](StockVison.ico)

**StockVison** est une application de bureau intelligente pour la gestion et l'analyse de stock, spécialement conçue pour les supermarchés et petites structures. Propulsée par l'IA **DeepSeek**, elle permet de dialoguer avec vos données en langage naturel, d'anticiper les ruptures et d'optimiser vos réapprovisionnements.

---

## 🚀 Fonctionnalités

- 📦 **Gestion complète des produits**  
  Ajoutez, modifiez, supprimez et suivez vos stocks en temps réel.

- 🤖 **Assistant IA DeepSeek**  
  Posez des questions en français (ex: « Quel est le produit le plus rentable ? »). Le chatbot exécute des requêtes SQL et vous répond instantanément.

- 📊 **Analyses avancées**  
  Identifiez les produits les plus rentables, visualisez les tendances de vente, détectez les articles en fin de vie et appliquez des remises dynamiques.

- 📁 **Import / Export JSON**  
  Importez vos données en masse ou exportez votre base pour sauvegarde.

- 🔐 **Authentification sécurisée**  
  Deux niveaux d'accès : Administrateur (gestion complète) et Employé (consultation + ventes).

- ☁️ **Base de données distante**  
  Hébergement MySQL sur Aiven – accès multi‑postes, sauvegardes automatiques, haute disponibilité.

---

## 🛠 Technologies utilisées

- **Langage** : Java (orienté objet)
- **Interface graphique** : JavaFX + Scene Builder
- **Base de données** : MySQL (hébergée sur Aiven)
- **IA conversationnelle** : DeepSeek
- **IDE** : IntelliJ IDEA Ultimate
- **Modélisation** : Lucidchart (diagrammes UML)

---

## 📥 Installation et exécution

### Prérequis
- Système d'exploitation : Windows, macOS ou Linux
- Java Runtime Environment (JRE) 11 ou supérieur
- Connexion Internet (pour la base de données distante)

### Téléchargement
Rendez-vous sur la page [Releases](https://github.com/samou2/StockVison/releases) et téléchargez l'exécutable correspondant à votre système :
- `StockVison-Setup-Windows.exe`
- `StockVison-macos.dmg`
- `StockVison-linux.AppImage`

### Lancement
1. Double‑cliquez sur l'exécutable.
2. La première ouverture peut prendre quelques secondes le temps d'établir la connexion à la base de données.
3. Utilisez les identifiants de démonstration (admin / admin) ou créez un compte employé.

> **Note** : Les identifiants de connexion à la base de données sont intégrés à l'application. Aucune configuration manuelle n'est nécessaire.

---

## 📖 Utilisation

1. **Connexion** : Saisissez votre nom d'utilisateur et mot de passe (admin ou employé).
2. **Tableau de bord** : Visualisez l'ensemble des produits, leur quantité et prix.
3. **Gestion** : Ajoutez, modifiez ou supprimez des produits via les boutons dédiés.
4. **Import/Export** : Utilisez les boutons "Importer JSON" et "Exporter JSON" pour gérer vos données en masse.
5. **Chatbot** : Cliquez sur l'icône du robot pour ouvrir l'assistant DeepSeek. Posez vos questions directement (ex: "Quels sont les produits avec un stock inférieur à 10 ?").
